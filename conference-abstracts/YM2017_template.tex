Пример оформления формул:
\begin{equation}
\label{eq_ivanov_1}
  {\bf A x} = {\bf b},
\end{equation}

\begin{equation}
\begin{array}{l}
\displaystyle
x = \frac{h^2}{3}\frac{\sigma}{\mu}\frac{\partial F}{\partial r}\\[3mm]
%
\displaystyle
\frac{\partial y}{\partial t} = -\frac{1}{r}\sqrt{1+\left(\frac{\partial h}{\partial r}\right)^2},\quad \mbox{где } h = g(r).
\end{array}
\label{eq_ivanov_2}
\end{equation}
Оформление ссылок на формулы (\ref{eq_ivanov_1}), (\ref{eq_ivanov_2}).

Примеры нумерованного списка:
\begin{enumerate}
\item неявная схема с приближенной LU-факторизацией для решения СЛАУ (\ref{eq_ivanov_1})
и явным заданием граничных условий;
\item неявная схема с точным обращением матрицы СЛАУ методами библеотеки Intel MKL для
разреженных матриц: PARDISO и FGMRES;
\item явная схема Рунге-Кутты по псевдовремени.
\end{enumerate}
